# Adding A Wiki Page

This is how you add a new wiki page to the Civic Dashboard Wiki!

<ol>
<li>Go to the branch dropdown on github, navigate to the 'View all branches'</li>
<li>In the view all branches click on 'New Branch' to create a new github branch</li>
<li>Enter your new branch name (i.e yourGithubUserName/DeputationsWiki), with the source being the 'main' branch</li>
<li>Click on the newly created branch under 'Your Branches'</li>
<li>Navigate to the contents/markdown folder</li>
<li>You can either click 'Add new file', and add a new file with the filename followed by .md file extension, please note the file conversion will not work with spaces so it will have to be a complete filename without spaces (i.e deputationsWiki.md) and not (deputations wiki.md).</li>
<li>Once you are done, you can commit your changes to the branch by clicking on the  'Commit Changes...'  button</li>
<li>You will then be prompted to enter a Commit Message, and a extended description for extra details.</li>
<li>Once you are ready, hit the commit changes button, and your markdown file will be committed to your branch on github.</li>
<li>Then you may create a new pull request by clicking on the pull requests tab, and create a new pull request, by clicking on 'New Pull Request'</li>
<li>In the 'Compare Changes' view, you must then select the base of 'main' branch, and then the source branch being the new branch you created i.e 'pliakhov/deputationswiki'.</li>
<li>Once both are selected, you can click the 'Create Pull Request', and your pull request will kick off the build process.</li>
<li>In the upload preview deployment(Once it is finished), you may view the preview URL, with your markdown document uploaded under the 'Wiki' tab.</li>
<li>In the build process once it's complete, under the 'All checks have passed', you can view the preview url under 'Preview Ready' with the url, with the URL available of the preview build, in which you can see a preview of the website with your markdown changes.</li>
<li>There you can click on the 'Wiki' tab with a link available to your document uploaded along with other wiki documents.</li>
</ol>
